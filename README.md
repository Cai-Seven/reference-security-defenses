# 企业级 NPM 供应链安全控制系统

防止恶意包进入 Build Pipeline 的完整解决方案。

## 核心特性

✅ **Build Runner 拦截** - 在 npm install 执行前验证所有包  
✅ **版本智能解析** - 支持 `^`, `~`, `>=` 等 semver 规范  
✅ **包年龄验证** - 防止<14天的新包（可配置）  
✅ **白名单管理** - 预批准的包和版本列表  
✅ **零开发侵入** - 完全透明，无需修改 package.json  
✅ **完整审计日志** - 所有检查操作可追溯  
✅ **离线 Fallback** - 支持缓存和本地白名单
